## Getting Started

### Prerequisites

- Flutter SDK (^3.8.1)
- Dart SDK
- Android Studio / VS Code

### Installation

1. Clone the repository
2. Install dependencies:
   ```
   flutter pub get
   ```
3. Configure your API settings in `lib/config.dart`
4. Run the app:
   ```
   flutter run
   ```
